TradingOS OKX windows

付费之后会自动启动策略文件 \n
可以通过策略文件config.json文件配置api和交易币种,
btc的usdt本位为BTC-USDT-SWAP,如果需要更换交易对,请自行修改, \n
例如ETH-USDT-SWAP,SOL-USDT-SWAP,请使用大写 \n
